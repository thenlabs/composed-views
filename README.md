## Installation.

Define in your `composer.json` file the next options:

```JSON
{
    "require": {
        "thenlabs/composed-views": "dev-master"
    },
    "minimum-stability": "dev",
    "prefer-stable": true
}
```

Then, run the command:

    $ composer update
