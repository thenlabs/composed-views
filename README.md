
# ComposedViews

## Instalación.

Cree o añada al archivo `composer.json` las siguientes dependencias:

```JSON
{
    "require": {
        "thenlabs/components": "dev-master",
        "thenlabs/class-builder": "dev-master",
        "thenlabs/cli": "dev-master",
        "thenlabs/composed-views": "dev-master"
    }
}
```
